Variables can still be named as reserved words with `$"name"` syntax.

### Reserved Words
* type
* raw-type
* let
* set
* if
* while
* match
* case
* loop
* interface
* implements
* opaque
* alias
* import
* zero
* atom
* suffix
* for
* sizeof
* bitsizeof
* alignof

### Reserved Types
* C-FFI
* FFI
